

# Median

[**Source code**](https://github.com/pola-rs/r-polars/tree/d562252dbb77de7e06ca3e6150d74a2c709763bc/R/lazyframe__lazy.R#L968)

## Description

Aggregate the columns in the LazyFrame to their median value.

## Usage

<pre><code class='language-R'>LazyFrame_median()
</code></pre>

## Value

A LazyFrame with one row

## Examples

``` r
library(polars)

pl$LazyFrame(mtcars)$median()$collect()
```
