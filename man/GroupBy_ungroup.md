

# GroupBy_ungroup

[**Source code**](https://github.com/pola-rs/r-polars/tree/f1aede4d7d7f090c98651365a4120a8232503a4d/R/group_by.R#L298)

## Description

Revert the group by operation.

## Usage

<pre><code class='language-R'>GroupBy_ungroup()
</code></pre>

## Value

DataFrame

## Examples

``` r
library(polars)

gb = pl$DataFrame(mtcars)$group_by("cyl")
gb
```
