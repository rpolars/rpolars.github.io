

# Get the first row of the DataFrame.

[**Source code**](https://github.com/pola-rs/r-polars/tree/8387e0a88c6889e6449b053999aada405c241066/R/dataframe__frame.R#L1028)

## Description

Get the first row of the DataFrame.

## Usage

<pre><code class='language-R'>DataFrame_first()
</code></pre>

## Value

A DataFrame with one row.

## Examples

``` r
library(polars)

pl$DataFrame(mtcars)$first()
```
