
# Get the last row of the DataFrame.

[**Source code**](https://github.com/pola-rs/r-polars/tree/53c7d964901ed4a019998e89aff8c6d44691d793/R/dataframe__frame.R#L1086)

## Description

Get the last row of the DataFrame.

## Usage

<pre><code class='language-R'>DataFrame_last()
</code></pre>

## Value

A DataFrame with one row.

## Examples

``` r
library(polars)

pl$DataFrame(mtcars)$last()
```
