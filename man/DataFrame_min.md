
# Min

[**Source code**](https://github.com/pola-rs/r-polars/tree/4c60e4ba5981c539b9639261157303d78f545b69/R/dataframe__frame.R#L1122)

## Description

Aggregate the columns in the DataFrame to their minimum value.

## Usage

<pre><code class='language-R'>DataFrame_min()
</code></pre>

## Value

A DataFrame with one row.

## Examples

``` r
library(polars)

pl$DataFrame(mtcars)$min()
```
