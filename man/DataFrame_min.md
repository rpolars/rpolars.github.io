

# Min

[**Source code**](https://github.com/pola-rs/r-polars/tree/97c09bc0a6fc3d166744dbddd037b49e8d8fc6c2/R/dataframe__frame.R#L1201)

## Description

Aggregate the columns in the DataFrame to their minimum value.

## Usage

<pre><code class='language-R'>DataFrame_min()
</code></pre>

## Value

A DataFrame with one row.

## Examples

``` r
library(polars)

pl$DataFrame(mtcars)$min()
```
