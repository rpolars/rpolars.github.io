## Description

Aggregate the columns in the LazyFrame to their minimum value.

## Usage

<pre><code class='language-R'>LazyFrame_min()
</code></pre>

## Value

A LazyFrame with one row

## Examples

``` r
library(polars)

pl$LazyFrame(mtcars)$min()$collect()
```
