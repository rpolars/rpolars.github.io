## Description

Compute the max value of the lists in the array.

## Usage

<pre><code class='language-R'>ExprList_max()
</code></pre>

## Format

function

## Value

Expr

## Examples

``` r
library(polars)

df = pl$DataFrame(values = pl$Series(list(1L, 2:3)))
df$select(pl$col("values")$list$max())
```
