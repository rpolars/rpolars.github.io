
# Get the first row of a LazyFrame

[**Source code**](https://github.com/pola-rs/r-polars/tree/4c60e4ba5981c539b9639261157303d78f545b69/R/#L)

## Description

Get the first row of a LazyFrame

## Usage

<pre><code class='language-R'>LazyFrame_first
</code></pre>

## Value

A LazyFrame with one row

## Examples

``` r
library(polars)

pl$LazyFrame(mtcars)$first()$collect()
```
